# Go Maps Examples

This folder demonstrates various scenarios for using maps in Go:

- **map_basic_usage.go**: Basic map creation, initialization, and access.
- **map_make_function.go**: Creating maps with the `make` function and setting capacity.
- **map_delete_operation.go**: Deleting elements from a map.
- **map_comma_ok_idiom.go**: Safe map access using the comma-ok idiom.
- **map_range_iteration.go**: Iterating over maps using `range`.

Each file contains a short prompt at the top describing its scenario.
